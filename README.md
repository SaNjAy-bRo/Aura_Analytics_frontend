# Aura Analytics Frontend

High-performance, conversion-focused landing page for Aura Analytics.

## Features
- **Modern Tech Stack**: Built with Tailwind CSS and HTML5.
- **Dark Mode Aesthetic**: Deep space theme with neon accents.
- **Glassmorphism**: Premium UI elements with backdrop blurs.
- **Responsive**: Fully optimized for mobile and desktop.
